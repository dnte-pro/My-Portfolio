My Projects

ShuttleHub

I built and deployed ShuttleHub, a full-stack badminton court booking platform. The frontend was developed in React.js and the backend in Node.js with MongoDB. I containerized the app using Docker and deployed it on AWS Elastic Beanstalk, enabling real-time booking with integrated payment support. This project helped me master containerization, cloud deployment, and CI/CD integration.
